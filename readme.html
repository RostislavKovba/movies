Задача Сделать каталог фильмов и вывести по ним данные. Требования 1. Основа 1.1. Ставим чистый WordPress 1.2. Поставить тему Twenty Twenty https://wordpress.org/themes/twentytwenty/ 1.3. Создать дочернюю тему для Twenty Twenty - все изменения кода делать только в дочерней теме 1.4. Добавить новый тип записи Фильмы для создания базы фильмов. Можно использовать расширение Custom Post Type Maker https://wordpress.org/plugins/custom-post-type-maker/ 1.5. К фильмам добавляем таксономии: Жанры, Страны, Год и Актеры. Чтобы классифицировать фильмы по жанрам и др условиям. Можно использовать тоже расширение ACF 1.6. Добавляем 3 текстовых поля к типу поста Фильмы: Постер (изображения), Стоимость сеанса и Дата выхода в прокат. Можно использовать расширение ACF http://wordpress.org/plugins/advanced-custom-fields/ По окончанию сгенерировать PHP код для для полей и зарегистрировать внутри дочерней темы https://www.advancedcustomfields.com/resources/register-fields-via-php/ 1.7. Забить данные фильмов и таксономий lorem ipsum на 5-7 фильмов, разных жанров, стран, стоимости и т д – чтобы убедиться что наполнение работает 2. Страницы 2.1. Список фильмов выводим на главной странице Для каждого фильма вывести : Название, Постер, Стоимость сеанса и Дата выхода в прокат. 2.2. Страница Single Фильма - Вывести всю информацию : Название, Постер, Стоимость сеанса и Дата выхода в прокат, Жанры, Страны, Год и Актеры 2.3. Страница таксономии - отображение списка фильмов по Жанру или Стране 3. Shortcode 3.1. Создать отдельно Shortcode для отображения 4 последних фильмов в любом месте сайта 4. Требования к Фронтенду (верстка) 4.1. Четких требований к виду и порядку вывода информации нет. Только минимально приемлемый вид отображения, респонсивность в рамках Bootstrap 5. Результат работы 5.1 Zip архив с кодом Проекта (или ссылка на GIT): WP + база данных в корне 5.2. Обязательно должны присутствовать только используемые Темы и Плагины 5.3.
